1.Definition 1.1.What is sentiment

● Sentiment = feelings

– Attitudes

– Emotions

– Opinions

● Subjective impressions, not facts

Generally, a binary opposition in opinions is assumed

● For/against, like/dislike, good/bad, etc.

1.Definition 1.2.What is sentiment analisys

-Using NLP( natural language processing ), statistics, or machine learning methods to extract, identify, or otherwise characterize the sentiment content of a text unit

- aims to determine the attitude of a speaker or a writer with respect to some topic


-Computational study of opinions, sentiments, evaluations, attitudes, appraisal, affects, views, emotions, subjectivity, etc., expressed in text.

2.What are the challenges?

– People express opinions in complex ways

– In opinion texts, lexical content alone can be misleading

– Intra-textual and sub-sentential reversals, negation, topic change common

– Rhetorical devices/modes such as sarcasm, irony, implication, etc.

3.How do we do it? Supervied methods

- Perceptron - The simplest algorithm to learn to separate two classes

-Calculates the dot product of the input and weight vectors

- Learns by example

- Outputs

- Threshold value Yes / No

- Real value (-1, 1)

A bit of a problem , works OK only for linear separable data

3.How do we do it? Supervied methods

- Backpropagation - the network learns from many inputs

Tries to minimize the error of the input weights

- works Good for non linear separable data

3.How do we do it? Supervied methods

- Naïve Bayes algorithm (max likelihood) - Find the most likely class c, given some features

Calculate the probability of the similar features

-small amount of training data

-training time is significantly smaller as opposed to alternative methods.

- works better for non linear separable data

3.How do we do it? Supervied methods

- Naïve Bayes Variations

-Multinomial Naive Bayes

-Binarized Multinomial Naive Bayes

-Bernoulli Naive Bayes.

* Note that each can deliver completely different results since they use completely different models.

4.Building Model step by step

Collect data

For example tweets:

4.Building Model step by step

Label Your Data

Once you have collected your training samples, you will need to pre-classify each sample with a label. A label is a string that you think best describes that example, for example: "happy", "sad", "on the fence".


"sad","Feeling kind of low...."

"excited", "OMG! Just had a fabulous day!"

"bored","Eating eggplant. Why bother?"


*Can have more than one label for instance in needed
